Unbox

Description

Array type for element types isomorphic to other element types with existing array support.

Primarily useful for unboxed arrays. As an example, define "instance Unbox (Sum Int) where type Unboxed (Sum Int) = Int" and use UArr i (Sum Int) as an unboxed array for Sum Int.
